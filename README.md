## Requirements

JDK 1.8 or greater


## How to use

- Gradle

Add jCenter repository on your build.gradle, and add implementation of dokument.

```groovy
repositories {
   jcenter()
}


dependencies {
  implementation 'br.eng.r2a.dokument:core:{dokument_version}'
}
```
